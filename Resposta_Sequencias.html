Respostas das Sequências

Respostas e Explicações das Sequências

a) 1, 3, 5, 7, _
Lógica: A sequência é composta por números ímpares consecutivos.
Próximo número: 9

b) 2, 4, 8, 16, 32, 64, __
Lógica: Cada número é o dobro do anterior (progressão geométrica com razão 2).
Próximo número: 128

c) 0, 1, 4, 9, 16, 25, 36, __
Lógica: A sequência é composta pelos quadrados dos números inteiros: 0², 1², 2², 3², 4², 5², 6², …
Próximo número: 49 (7² = 49)

d) 4, 16, 36, 64, __
Lógica: A sequência é composta pelos quadrados dos números pares: 2², 4², 6², 8², …
Próximo número: 100 (10² = 100)

e) 1, 1, 2, 3, 5, 8, __
Lógica: Sequência de Fibonacci, onde cada número é a soma dos dois anteriores.
Próximo número: 13 (5 + 8 = 13)

f) 2, 10, 12, 16, 17, 18, 19, __
Lógica: A sequência alterna entre adições maiores e menores. Aumentos de +8, +2, +4, +1, +1, +1…
Próximo número: 20
